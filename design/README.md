# HarmonyOS 智能输入法设计目录

本目录用于沉淀这款鸿蒙原生输入法的完整方案，目标能力包括：

- 中文拼音输入
- 英文输入
- 语音输入
- 服务端 LLM 润色与口语化清洗
- 词库、配置、隐私安全与测试验收

配套样例：

- `config/`：配置、子类型、策略、词库清单样例
- `samples/`：拼音词库、英文词库、ASR 热词、用户词条样例
- `sensevoice/`：sherpa-onnx SenseVoice 最小版端侧语音识别方案

设计约束说明：

- 当前工程使用 HarmonyOS `6.0.2(22)` 工具链。
- 输入法系统能力、面板、输入客户端、子类型、编辑框扩展配置等约束，已按本机 SDK 中的 `IMEKit` 和 `CoreSpeechKit` 声明做了校验。
- 个别与 `module.json5` 元数据键名相关的配置，因 SDK 声明文件不直接暴露 profile 键名，文档中会明确标注“需以落地时官方文档为准”的点。
- 中文拼音引擎方案已补充为“`librime` 核心引擎 + Harmony 原生桥接层 + ArkTS 面板与策略层”的实现路线。
